STM_Provisioning example is used to write DAC/PAI certificates & DAC public key 
in Factory Data and DAC private key in CKS.
It is part of the Factory Data provisioning procedure depicted in 
more details in the wiki:
https://wiki.st.com/stm32mcu/wiki/Connectivity:Matter_device_factory_data_provisioning

@par Keywords

Matter,DAC,PAI,CKS,Provioning

@par Directory contents 

 
  Using Matter v1.4.0.0 SDK.
  
 
@par Hardware and Software environment

  - This example has been tested with an STMicroelectronics WB5MM-DK board 
    
@par How to use it ? 
refer to wiki


Minimum requirements for the demo:
- 1 STM32WB5MM-DK board.
