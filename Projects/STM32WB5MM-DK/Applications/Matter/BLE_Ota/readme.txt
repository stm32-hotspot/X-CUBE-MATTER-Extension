@par Application Description

OTA implementation to download a new image into the user flash.

@note This Application is embedded in Matter-App that support SFSFU.
      It is used to update M0 and/or M4 firmwares over BLE.
      See Release Notes for supported features.
      See Wiki article how to demonstrate BLE_Ota.
